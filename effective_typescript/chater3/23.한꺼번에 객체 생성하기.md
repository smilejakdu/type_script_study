# 자바스크립트 객체 생성

```js
const pt = {};
pt.x = 3;
pt.y = 4;
```
하지만 타입스크립트는 이렇게 생성이 불가능하다.

```ts
const pt = {};
pt.x = 3;
// ~ '{}' 형식에 'x' 속성이 없습니다.
pt.y = 4;
// ~ '{}' 형식에 'y' 속성이 없습니다.
``` 

이 문제는 객체를 한번에 만들면 해결이 된다.

아니면 타입 단언문을 사용해서도 해결 할수도 있다.

```ts
interface Point { x: number; y: number; }
const pt: Point = {
  x: 3,
  y: 4,
};
```

혹은 

```ts
const pt = {} as Point;
pt.x = 3;
pt.y = 4;
```
이렇게 할수도있지만 되도록 객체를 한꺼번에 만드는것이 더 좋습니다.
